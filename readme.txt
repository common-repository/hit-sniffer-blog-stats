=== Hit Sniffer Live Blog Analytics ===
Contributors: San Arena
Tags: live,stats, statistics,hits,hit, web, visitor,blog
Requires at least: 1.5
Tested up to: 4.6.1
Stable tag: 2.12

Hit Sniffer was a powerful real time website visitor activity tracker. 

== Description ==

Hit Sniffer service has shutdown on September 2016
This plugin now serve as a memorial and soon to be deleted.

Those who look for similiar services, they can use Hit Steps.
